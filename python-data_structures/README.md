# Python - Data Structures

This repository contains Python scripts demonstrating lists and tuples usage.

## Files Description

* `0-print_list_integer.py` - Prints list integers
* `1-element_at.py` - Retrieves element from list
* `2-replace_in_list.py` - Replaces element at position
* `3-print_reversed_list_integer.py` - Prints reversed list
* `4-new_in_list.py` - Replaces element without modifying original
* `5-no_c.py` - Removes characters 'c' and 'C'
* `6-print_matrix_integer.py` - Prints integer matrix
* `7-add_tuple.py` - Adds two tuples
* `8-multiple_returns.py` - Returns string length and first char
* `9-max_integer.py` - Finds largest integer
* `10-divisible_by_2.py` - Finds multiples of 2
* `11-delete_at.py` - Deletes item at position
* `12-switch.py` - Switches variable values
* `13-is_palindrome.c` - Checks palindrome in linked list
* `lists.h` - C header file
